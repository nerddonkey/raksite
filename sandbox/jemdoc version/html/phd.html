How to apply to the ANU Doctor of Philosophy (PhD) Program

Follow the application process described in Graduate Research

  1. Perform the Self Assessment

  2. If you pass the first step then read Pre-application process

  3. Send an email to me that

    1. is clear and coherent

    2. indicates where you were ranked in your year, prizes you have received and any papers you have written

    3. has only PDF attachments (not Word doc)

    4. contains only academically pertinent information

  4. Don't expect a response (as I get numerous such emails each day) if you fail to heed the above requirements
